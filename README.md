# 模糊视觉辅助问答系统

一个面向健康场景的多模态视觉问答系统前端项目，支持图片上传、摄像头拍照、问题输入、结果展示、历史记录查看，以及用户端 / 管理端分离的系统界面设计。

当前仓库以前端实现为主，后端接口已预留，现阶段使用 Mock 数据完成整体交互流程，便于后续团队成员直接接入真实模型与服务。

## 项目简介

模糊视觉辅助问答系统主要用于对用户上传或拍摄的图像进行辅助理解和问答，适用于健康相关场景，例如：

- 药品包装识别
- 标签文字读取
- 日用品用途辅助判断
- 场景风险提醒
- 模糊图像重拍引导

系统采用“用户端 + 管理端”双端结构：

- 用户端：面向普通用户，提供图像上传、拍照识别、问题输入、结果查看、历史记录等功能
- 管理端：面向管理人员，提供数据统计、用户管理、问答记录、模型管理、风险与安全、日志管理等功能
